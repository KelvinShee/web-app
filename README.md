# Web Application Project

This repository contains a Flask-based web application deployed using Docker, Kubernetes (Minikube), and a CI/CD pipeline with GitHub Actions.

## Files
- `app.py`: Flask application code.
- `Dockerfile`: Docker configuration for building the app image.
- `web-app.yaml`: Kubernetes manifest for the web app deployment.
- `postgres.yaml`: Kubernetes manifest for the PostgreSQL database.

## Setup
- Clone the repository: `git clone https://github.com/KelvinShee/web-app.git`
- Follow the instructions in the presentation for deployment.
